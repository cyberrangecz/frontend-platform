# CyberRangeᶜᶻ Platform

This is the main runnable application which utilizes all of the libraries to provide the final functionality of the CyberRangeᶜᶻ Platform. It is the app that gets built and deployed. The application configures all necessary resources, provides the home screen component, login screen component, and routing configuration.

The application uses the configuration file located at `public/config.json`, which defines base paths for various services, authentication settings, and other platform configurations when the development server is running.

## Running the App

Run `nx serve cyberrangecz-platform` to start the development server.
