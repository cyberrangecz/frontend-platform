# CyberRangeᶜᶻ Platform Utils Components

This library contains utility Angular components used across the platform for common UI elements.

- **Loading Dialog**: Component for displaying loading dialog with customizable options.
- **File Upload Dialog**: Component for handling file upload dialogs.
- **Table Date Cell**: Component for displaying and synchronizing dates in table cells.
- **Table State Cell**: Component for displaying state information in table cells.
- **Table Countdown**: Component for countdown timers in table cells.
- **Logo Spinner**: Component with CRCZP logo animated as a loading spinner.
